# Radio Streaming

Web-based radio player with background audio streaming, station browsing, and playback controls.

## Features

- Live radio station streaming
- Station search and favorites
- Background playback support
- Playback controls (play, pause, volume)
- Station metadata display

## Tech Stack

TypeScript, React, Web Audio API, REST APIs

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/Radio-Streaming.git
cd Radio-Streaming
npm install
```

### Usage

```bash
npm run dev
```
